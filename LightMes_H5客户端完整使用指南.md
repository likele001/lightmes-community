## 应用概述

### 🎯 应用定位

LightMes H5 是专为**客户/会员**设计的移动端应用，支持：
- ✅ **在线下单** - 浏览产品 → 选择型号规格 → 下单
- ✅ **进度查询** - 实时查看订单生产进度
- ✅ **订单管理** - 查看订单历史、详情、交期
- ✅ **对账单** - 查看销售对账单、账款记录
- ✅ **消息推送** - 接收订单更新、生产进展通知

### 📱 应用特点

- **响应式设计**：自适应手机屏幕（iPhone、Android）
- **离线可用**：支持部分功能离线浏览（如历史订单）
- **快速加载**：优化移动网络场景
- **简洁易用**：极简设计，快速上手
- **实时更新**：订单进度实时推送

### 🔗 访问方式

```
Web URL: https://your-domain/lightmes/h5
或 小程序版本: 未来支持
```

## 登录与认证

### 💻 登录步骤

1. **打开应用**
   - 在手机浏览器输入应用 URL
   - 或扫描企业提供的二维码

2. **进入登录页**
   - 显示登录表单
   - 输入**用户名**（手机号或账号）
   - 输入**密码**

3. **验证身份**
   - 点击「登录」按钮
   - 系统验证账户信息
   - **首次登录**可能需要完善用户资料

4. **进入应用首页**
   - 登录成功后自动跳转到首页
   - 显示个人信息和快速菜单

### ⚠️ 注意事项

- **账户由企业分配**，联系销售或客服获取登录凭证
- **密码保管**：不要告诉他人
- **忘记密码**：点击「忘记密码」 → 输入手机号 → 接收验证码 → 重置密码
- **账号关联**：一个客户账户对应一个公司
- **多终端登录**：支持多个设备同时登录（新登录会挤掉旧登录）
- **会话超时**：15分钟未操作会自动退出，重新登录即可
- **首次登录提示**：可能需要修改初始密码

### 🔐 安全建议

- ✅ 使用**强密码**（包含大小写字母、数字、特殊字符）
- ✅ 定期**更换密码**（建议每3个月）
- ✅ 不要在公共计算机上勾选「记住我」
- ✅ 使用完毕后**点击退出登录**
- ✅ 如发现异常登录，立即修改密码或联系客服

## 客户下单

### 📦 下单入口

**路径**: 底部导航「下单」或 首页「我要下单」

### ✅ 完整下单流程

#### **第一步：浏览产品**

1. 进入下单页面，显示**两个选项卡**：
   - **产品浏览**（默认）
   - **我的订单**（历史订单）

2. 在「产品浏览」页查看产品信息：
   - **产品列表**显示所有可下单的产品
   - 每个产品显示：
     - 产品名称
     - 产品图片（如有）
     - 产品编码/分类
     - 简短描述

3. **按分类过滤**（可选）：
   - 点击「产品分类」下拉菜单
   - 选择要查看的产品类别
   - 列表自动更新

4. **搜索产品**（可选）：
   - 在搜索框输入产品名称或编码
   - 支持模糊搜索
   - 实时显示匹配结果

#### **第二步：选择型号与规格**

1. 点击要购买的产品，进入**型号选择页**
2. 显示该产品的所有可用型号：
   - 型号编码
   - 颜色（如有）
   - 材料（如有）
   - 规格（如有）
   - 库存状态（仅显示可用型号）
   - 参考单价（如配置）

3. **选择一个型号**：
   - 点击型号卡片
   - 系统高亮选中

#### **第三步：输入订单信息**

1. 在下方表单中输入：
   - **订单数量**（必填）：
     - 输入需要购买的数量
     - 建议输入合理的数量（最小1件）
   
   - **交期**（可选）：
     - 点击「选择交期」
     - 在日期选择器中选择需要的交期
     - 默认为当前日期+7天
     - 交期不能早于今天
   
   - **特殊要求/备注**（可选）：
     - 输入特殊说明
     - 如：「需要加急」、「特殊包装」等
     - 字数限制200字

2. 显示订单预估金额（如配置产品单价）

#### **第四步：确认并提交**

1. 检查订单信息：
   - 产品型号是否正确
   - 订单数量是否准确
   - 交期是否满足需要

2. 点击「提交订单」
   - 按钮变为加载状态
   - 显示「提交中...」

3. 提交成功：
   - 页面显示「订单已提交」提示
   - 跳转到**订单详情页**
   - 显示新订单的编号和状态

### ⚠️ 下单注意事项

- **确认产品型号**：
  - 确保选择的是正确的颜色/材料/规格
  - 如不确定，可以在备注里说明具体需求

- **交期合理性**：
  - 交期不要太紧（给生产留充分时间）
  - 一般建议至少**提前7天**下单
  - 紧急订单需要与工厂联系确认可行性

- **数量准确**：
  - 仔细确认订单数量
  - 提交后**订单状态为「草稿」时可以修改**
  - 一旦确认就无法自行修改，需要联系客服

- **备注说明**：
  - 有特殊要求一定要写清楚（如包装、标签、送货方式等）
  - 避免理解偏差

- **网络问题**：
  - 如提交失败，稍后重试
  - 避免重复点击「提交」按钮（可能重复下单）

### 🔄 订单状态说明

| 状态 | 含义 | 说明 |
|------|------|------|
| 草稿 | 订单已保存但未确认 | 可以修改或删除 |
| 已确认 | 工厂已确认订单 | 开始安排生产 |
| 生产中 | 正在生产加工 | 可查看生产进度 |
| 已完成 | 生产完成 | 等待发货 |
| 已发货 | 产品已发出 | 可追踪物流 |
| 已取消 | 订单已取消 | 可咨询取消原因 |

## 订单管理

### 📋 订单列表

**路径**: 底部导航「订单」或 首页「订单列表」

#### **查看所有订单**

1. 进入订单列表页面
2. 显示所有历史订单（按时间倒序）
3. 每个订单显示：
   - 订单号（点击可查看详情）
   - 产品名称/型号
   - 订单数量
   - 订单状态（用不同颜色标记）
   - 交期
   - 下单时间
   - 预计完成时间

#### **筛选订单**

1. **按状态筛选**：
   - 点击「筛选」按钮或顶部状态标签
   - 选择要查看的状态：
     - 全部 | 草稿 | 已确认 | 生产中 | 已完成 | 已发货

2. **按时间筛选**：
   - 选择时间范围（如本月、上月、自定义日期）
   - 列表自动更新

3. **搜索订单**：
   - 在搜索框输入订单号或产品名
   - 实时显示匹配的订单

#### **订单卡片操作**

点击订单卡片可以：
- 📖 查看完整订单详情
- 📍 查看订单生产进度
- 📞 联系客服（快速拨号或发消息）
- 🔖 标记订单（星标、备注）
- 📥 下载订单文档（如有）

### 📄 订单详情页

**打开方式**: 在订单列表点击订单卡片

#### **显示信息**

1. **基本信息**：
   - 订单号
   - 订单状态（大字显示）
   - 下单时间
   - 交期

2. **产品信息**：
   - 产品名称/编码
   - 产品型号（颜色、材料、规格）
   - 订单数量
   - 单位

3. **预估/实际成本**（如配置）：
   - 产品单价
   - 订单总额
   - 已支付金额（如有）
   - 待支付金额

4. **送货信息**：
   - 收货地址
   - 送货方式（自提/快递/物流）
   - 送货时间

5. **备注信息**：
   - 下单时的特殊备注
   - 工厂回复或标记

#### **订单操作**

- **查看进度** 🔍：点击「查看生产进度」
- **修改订单**（仅草稿状态）：修改数量或交期
- **删除订单**（仅草稿状态）：确认后删除
- **打印订单**：生成 PDF 可打印
- **分享订单**：分享给企业内部同事
- **联系客服**：快速联系工厂客服

## 订单进度追踪

### 📍 进度查询入口

**路径**: 底部导航「追踪」或 首页「订单追踪」或 订单详情「查看进度」

### ✅ 进度页面结构

```
┌────────────────────────────────┐
│ 订单号: ORD202605001            │
│ 状态: 生产中 | 预计交期: 5.28  │
└────────────────────────────────┘
│
│ 进度时间轴（Timeline）
│ ├─ ✓ 2026-05-19 10:30 订单已确认
│ ├─ ✓ 2026-05-19 11:00 生产已启动
│ ├─ ⏳ 2026-05-20 工序: 冲压（40%）
│ ├─ ⏳ 2026-05-21 工序: 焊接（0%）
│ ├─ 2026-05-22 工序: 组装
│ └─ 2026-05-28 订单完成/发货
│
│ 工序详情（可展开）
│ 当前工序: 冲压
│ ├─ 派工人数: 5人
│ ├─ 已完成: 400件/1000件 (40%)
│ ├─ 预计完成: 2026-05-20 15:00
│ └─ 进度图表（实时更新）
│
│ 操作按钮
│ ├─ 刷新进度 🔄
│ ├─ 分享进度 📤
│ └─ 询问工厂 💬
│
└────────────────────────────────┘
```

### 📊 进度信息详解

#### **订单概览**

- **订单号**：唯一的订单编号
- **订单状态**：当前状态（生产中/已完成等）
- **进度百分比**：整体完成率（0-100%）
- **交期提醒**：
  - 🟢 绿色：进度正常
  - 🟡 黄色：进度略滞后
  - 🔴 红色：进度严重滞后，可能无法按期完成

#### **工序时间轴**

显示该订单的每个工序及其状态：

| 状态 | 图标 | 说明 |
|------|------|------|
| 已完成 | ✓ | 工序已完成，显示完成时间 |
| 进行中 | ⏳ | 工序正在进行中，显示进度百分比 |
| 待进行 | ◯ | 工序还未开始，显示预计开始时间 |

#### **工序详情**

点击任意工序卡片可查看：
- **工序名称**：如 冲压、焊接、组装
- **派工人数**：该工序分配的员工数
- **完成进度**：已完成数量 / 总数量 (百分比%)
- **预计完成时间**：该工序预计何时完成
- **质检状态**：是否通过质检（如显示）

### 🔄 进度更新机制

- **自动刷新**：页面每2分钟自动更新一次最新进度
- **手动刷新**：点击「刷新」按钮立即更新
- **实时推送**（可选）：
  - 如启用消息推送，工序完成时会收到通知
  - 在消息中心可查看历史通知

### ⚠️ 进度查询注意事项

- **延迟**：显示数据可能延迟1-2分钟（系统处理时间）
- **提前告知**：如需了解实时进展，**提前3-5天与工厂沟通**
- **异常情况**：
  - 如进度长时间不更新，可能出现异常
  - 建议直接联系工厂客服
- **多个订单**：
  - 如同时有多个订单在生产
  - 可以逐个查看各订单进度

   - 总金额

   - 待结款金额
   - 对账状态（未对账/已对账）

3. **筛选与搜索**：

#### **对账单内容**

1. **对账周期**：
   - 起始日期 ~ 结束日期
   - 通常按自然月或按客户要求的周期

2. **金额统计**：
   - **订单总额**：本周期所有订单的总金额
   - **已支付**：客户已实际支付的金额
   - **待支付**：客户还需支付的金额

3. **订单明细表**：
   - 订单号
   - 产品名称
   - 数量
   - 单价 / 总价
   - 交期
   - 订单状态

4. **收款信息**（如配置）：
   - 银行账号（脱敏显示）
   - 账户开户名
   - 联行号

### 📄 对账单操作

- **下载 PDF**：点击「下载」生成对账单 PDF，可打印或存档
- **申请发票**：点击「申请发票」，提交发票申请表
- **对账确认**：如对账单无误，点击「对账确认」
- **提出异议**：如数据不符，点击「提出异议」并说明原因
- **联系财务**：快速拨号或发消息给财务联系人

### ⚠️ 对账注意事项

- **及时确认**：收到对账单后**尽快审核确认**
- **核对无误**：
  - 确认订单号、数量、金额是否准确
  - 检查已支付和待支付的区分是否正确
- **异议处理**：
  - 如发现错误，立即提出异议
  - 提供具体的订单号和异议原因
  - 工厂会在3-5个工作日回复
- **支付及时**：
  - 尽量按约定的账期支付
  - 避免逾期

## 消息通知

### 🔔 消息中心

**路径**: 底部导航「我的」→ 「消息中心」或 首页「消息中心」

### 📬 消息类型

应用会推送以下几类消息：

| 消息类型 | 触发条件 | 示例 |
|--------|--------|------|
| **订单确认** | 工厂确认了您的订单 | 订单ORD001已确认，预计5.28交期 |
| **生产开始** | 订单开始生产 | 订单ORD001已开始生产 |
| **工序完成** | 完成一个关键工序 | 订单ORD001冲压工序已完成 |
| **生产完成** | 全部工序完成 | 订单ORD001生产已完成，准备发货 |
| **已发货** | 订单已发出 | 订单ORD001已发货，快递号为SF123456 |
| **对账通知** | 新的对账单生成 | 您的2026年5月对账单已生成 |
| **发票通知** | 发票已开具 | 发票已开具，可在对账单中下载 |
| **系统通知** | 平台更新、维护等 | 系统将于今晚22:00-23:00维护 |

### 📲 消息管理

#### **查看消息**

1. 进入消息中心
2. 显示所有消息列表（按时间倒序）
3. 每条消息显示：
   - 消息类型图标
   - 消息标题和摘要
   - 接收时间（相对时间：1小时前、3天前等）
   - 已读/未读状态

#### **消息操作**

- **点击消息**：查看完整内容
- **标记已读**：长按消息或点击批量操作
- **删除消息**：滑动消息或选择后删除
- **标记为星标**：重要消息可星标方便查找

#### **消息设置**

进入「设置」可以配置：
- **推送开关**：是否接收 App 推送通知
- **消息类型**：选择想要接收的消息类型
- **推送时间段**：设置不打扰的时间
- **声音和震动**：是否启用声音和震动提醒

### 💡 消息通知最佳实践

- ✅ **启用推送**：及时获知订单动态
- ✅ **标记星标**：重要消息快速定位
- ✅ **定期检查**：即使禁用推送也要定期查看
- ✅ **检查邮箱**：重要通知也会发邮件
- ❌ **勿忽视红点**：未读消息通常表示重要信息

## 常见问题

### Q1: 忘记登录密码怎么办？

**A**: 
1. 点击登录页的「忘记密码」
2. 输入注册时的手机号
3. 接收验证码短信
4. 输入验证码和新密码
5. 点击「重置密码」完成

如仍无法自助重置，联系客服，需要验证身份后由人工重置。

---

### Q2: 提交订单后可以修改吗？

**A**: 
- ✅ **可以修改**（状态为「草稿」时）：
  - 进入订单详情
  - 点击「编辑」
  - 修改数量或交期
  - 保存

- ❌ **不能修改**（状态为「已确认」后）：
  - 需要作废原订单
  - 重新新建订单
  - 或联系客服说明

---

### Q3: 订单何时可以下单？

**A**: 
- 一般来说，**随时可以下单**
- 建议**至少提前7天**下单，给工厂充分的生产时间
- 如需紧急订单（如2-3天交期），**提前电话咨询工厂**是否可行
- 如工厂已通知停工休假，则无法接单

---

### Q4: 如何知道订单是否按时完成？

**A**: 
有三种方式查看：
1. **消息推送**：生产完成时会自动推送通知
2. **进度查询**：进入「订单追踪」实时查看进度
3. **订单列表**：订单卡片上显示预计完成时间和当前状态

---

### Q5: 进度显示「延迟」是什么意思？

**A**: 
- 表示当前生产进度可能**无法按时完成**
- 原因可能是：
  - 物料延迟
  - 产能不足
  - 质检不通过需要返工
- **建议立即联系工厂客服**了解具体情况和解决方案

---

### Q6: 产品图片为什么看不到？

**A**: 
- 工厂可能**未上传产品图片**
- 或您的网络不稳定导致加载失败
- **解决方法**：
  - 刷新页面重试
  - 检查网络连接
  - 联系客服获取产品资料

---

### Q7: 对账单金额与我的账目不符怎么办？

**A**: 
1. 仔细核对对账单中的每个订单号和金额
2. 检查是否有已取消的订单被包含或遗漏
3. 确认已支付金额是否计算正确
4. 如仍不符，点击「提出异议」说明情况
5. 工厂财务会在3-5个工作日回复

---

**A**:

## 🎯 使用建议

### ✅ DO（应该做）

- ✅ **定期查看进度**：避免错过重要的生产节点
- ✅ **及时提出异议**：如对账单或订单信息有问题
- ✅ **启用消息推送**：不会错过重要通知
- ✅ **妥善保管密码**：账号安全很重要
- ✅ **提前下单**：至少提前一周，给工厂充分时间

### ❌ DON'T（不应该做）

- ❌ **不要频繁刷新进度**：会增加服务器压力，每2分钟自动更新已足够
- ❌ **不要在最后一刻下单**：可能无法按期完成
- ❌ **不要使用简单密码**：比如「123456」、「password」
- ❌ **不要在公共设备保留登录状态**：登出后再离开
- ❌ **不要提交过大的文件**：订单备注或对账单下载可能卡顿
