## 应用概述

### 🎯 应用定位

LightMes H5 员工端是为**工厂一线员工**设计的移动端应用，支持：
- ✅ **查看任务** - 实时查看派给自己的生产任务
- ✅ **扫码报工** - 使用手机摄像头扫描二维码快速报工
- ✅ **查看工资** - 实时查看预估工资和月度工资条
- ✅ **考勤打卡** - 支持手机打卡签到/签退
- ✅ **消息提醒** - 接收任务派工、报工审核等实时推送

### 📱 应用特点

- **极简界面**：大字体、大按钮，方便在车间快速操作
- **离线模式**：支持任务离线查看
- **快速报工**：扫码一秒钟完成任务识别
- **实时反馈**：报工后立即显示预估工资
- **移动友好**：完全适配手机屏幕，支持竖屏操作

### 🔗 访问方式

```
Web URL: https://your-domain/lightmes/h5
或二维码：工厂 HR 会发放员工专用二维码
```

## 登录与身份认证

### 💻 首次登录

1. **打开应用**
   - 在手机浏览器输入应用 URL
   - 或扫描工厂发放的二维码

2. **进入登录页**
   - 显示用户名和密码输入框
   - 输入**用户名**（通常是员工工号或手机号）
   - 输入**密码**

3. **点击登录**
   - 系统验证账户
   - 首次登录可能要求设置安全问题或修改初始密码

4. **成功登录**
   - 自动跳转到首页
   - 显示个人信息和今日任务

### ⚠️ 登录注意事项

- **账户由 HR 分配**：联系人力资源部获取初始账户
- **初始密码保管**：工厂分配的初始密码要妥善保管
- **首次登录必改密码**：出于安全考虑，第一次登录必须修改初始密码
- **忘记密码**：
  - 点击「忘记密码」
  - 输入注册手机号
  - 接收短信验证码
  - 设置新密码
  - 若仍无法重置，咨询 HR
- **会话安全**：
  - 工作完毕记得「退出登录」
  - 不要在公共设备上保留登录状态
  - 15分钟无操作会自动退出

### 🔐 安全建议

- ✅ 设置**强密码**（大小写混合、数字、特殊字符）
- ✅ **定期修改密码**（每月至少一次）
- ✅ 不要告诉他人密码
- ✅ 使用完毕**必须退出登录**
- ✅ 发现账户异常立即联系 HR

## 扫码报工

### 📷 报工入口

**路径**: 任务列表「扫码报工」或 底部导航「报工」或 首页「扫码报工」

### ✅ 完整报工流程

#### **第一步：扫码识别任务**

有两种方式识别任务：

**方式1：扫二维码（推荐）**
1. 进入报工页面
2. 点击「打开摄像头」
3. 对准任务标签上的**二维码**
4. 系统自动识别任务
5. 显示任务信息：
   - 任务编码
   - 产品型号
   - 工序名称
   - 派工数量

**方式2：手动输入任务码**
1. 如果没有二维码或无法扫描
2. 手动输入**任务编码**（在任务卡片上有显示）
3. 点击「查询」
4. 系统加载任务信息

#### **第二步：输入报工数据**

1. **填写合格数量**（必填）：
   - 输入本次完成的**合格品数量**
   - 必须 ≤ 派工数量

2. **填写不良数量**（必填）：
   - 输入本次完成的**不良品数量**
   - 如果全部合格，填 0

3. **验证数量**：
   - 系统自动验证：合格数 + 不良数 ≤ 派工数
   - 如不合法，会提示错误

```
例1: 派工100件
  输入: 合格90件, 不良5件
  ✓ 合法 (90+5=95 ≤ 100)

例2: 派工100件
  输入: 合格100件, 不良5件
  ✗ 不合法 (100+5=105 > 100)
  需要修正
```

#### **第三步：上传质量证据（可选但推荐）**

质量证据非常重要，班组长审核时需要查看：

1. **拍摄照片**：
   - 点击「拍照」按钮
   - 拍摄完工产品的照片
   - 建议拍摄**多个角度**（正面、侧面、细节）
   - 可上传1-5张照片

2. **录制视频**（可选）：
   - 点击「录像」按钮
   - 录制工作过程视频（10-30秒）
   - 展示工序完成情况

3. **选择上传的文件**：
   - 系统会显示已选择的文件列表
   - 可以删除不想要的文件

#### **第四步：填写备注**

在「备注」框输入特殊说明（可选）：
- 工作中遇到的问题
- 特殊情况说明
- 如：「机器故障延迟2小时」「材料有问题，已通知班组长」

#### **第五步：确认并提交**

1. **再次检查数据**：
   - 任务编码是否正确
   - 合格数/不良数是否准确
   - 照片是否清晰

2. **点击「提交报工」**：
   - 按钮变为「提交中...」
   - 系统上传数据和文件

3. **提交成功反馈**：
   ```
   ✓ 报工成功！
   
   任务: TSK20260501 冲压工序
   合格: 90件
   不良: 5件
   
   📊 预估工资: ¥180.00
   （基于工序单价×合格数）
   ```

### 💰 预估工资显示

报工成功后，系统会立即计算并显示**预估工资**：

```
预估工资计算:
  工序单价 ¥2.00/件
  × 合格数 90件
  = 预估工资 ¥180.00

⚠️ 说明:
这是基于当前工价的初步估算
最终工资需要班组长初审和QC终审通过
审核通过后自动计入月工资
```

### ⚠️ 报工注意事项

- **准确填写数量**：
  - 合格数是经过质检的、可以销售的产品
  - 不良数是有缺陷、需要返工或报废的产品
  - 虚报会影响工资和考核

- **上传质量证据**：
  - **拍照很重要**，班组长和QC要查看
  - 照片要清晰，能看清产品的细节
  - 如无法上传（网络问题），先本地保存，待网络恢复后补传

- **备注要清楚**：
  - 有特殊情况要说明
  - 如：「机器故障」「材料不符」等
  - 便于班组长理解和审核

- **报工后无法修改**：
  - 一旦提交，自己无法改动
  - 如数字填错了，可以请班组长「驳回」
  - 班组长驳回后，重新报工即可

- **避免重复提交**：
  - 不要连续点击「提交」按钮
  - 可能导致重复报工

### 📹 多媒体上传建议

| 文件类型 | 格式 | 大小限制 | 建议 |
|--------|------|--------|------|
| **照片** | JPG/PNG | 5MB 以内 | 3-5张，多角度 |
| **视频** | MP4 | 10MB 以内 | 10-30秒，体现工序 |

💡 **压缩建议**：
- 手机拍摄的照片通常较大，建议压缩后上传
- 视频可用手机自带剪辑工具剪短
- 不要上传超大文件，容易上传失败

### 💵 工资组成

#### **基础工资**
- **来源**：该月所有**审核通过的报工**
- **计算**：Σ(报工数量 × 工序工价)
- **说明**：按件数计算，多劳多得

#### **补贴**
- **全勤奖**：如当月无迟到/旷工
- **岗位津贴**：如担任班组长等职务
- **其他补贴**：工厂额外奖励

#### **扣款**
- **迟到扣款**：每次迟到扣款金额
- **旷工扣款**：缺席一天扣款
- **其他扣款**：罚款等

#### **应发工资**
```
应发工资 = 基础工资 + 补贴 - 扣款
```

#### **支付情况**
- **已支付**：工资发放到手的金额
- **待支付**：还没发放的部分（通常为0）

### ❓ 工资疑问

- **基础工资与实际报工不符**：
  - 基础工资只包含**已审核通过的报工**
  - 待审核的报工暂未计入
  - 被驳回的报工不计入
  
- **为什么有些报工没计入**：
  - 班组长或QC还在审核，状态为「待初审」或「待终审」
  - 报工被驳回了，需要重新报工
  
- **补贴/扣款不对**：
  - 联系班组长或财务解释
  - 通过「联系客服」快速沟通

### 🔍 常见工资问题

**Q: 工资条里的基础工资与我预估的不同？**
- A: 预估工资是报工时立即计算的
- 最终工资需要班组长初审 → QC终审通过
- 被驳回的报工不计入最终工资
- 所以最终工资通常会低于预估工资

**Q: 补贴没有拿到，怎么办？**
- A: 补贴是工厂根据政策手动添加的
- 如月初工资条仍无补贴，咨询班组长或财务
- 可能是审核还未完成

**Q: 工资被扣款了，是什么原因？**
- A: 通常是迟到、旷工等原因
- 查看工资条的「扣款说明」
- 如有疑问，问班组长为什么被扣款

### ✅ 打卡流程

#### **签到打卡**

1. **进入打卡页**
   - 显示当前时间
   - 显示上班时间（如08:00）

2. **点击「签到」按钮**
   - 系统记录打卡时间
   - 对比上班时间判断是否迟到
     - ≤08:00：准时 ✓
     - >08:00：迟到 ⚠️

3. **打卡成功**
   ```
   ✓ 签到成功
   打卡时间: 2026-05-19 07:58
   状态: 准时
   ```

#### **签退打卡**

1. **工作结束时**
   - 点击「签退」按钮
   - 系统记录下班时间
   - 显示工作时长

2. **打卡成功**
   ```
   ✓ 签退成功
   打卡时间: 2026-05-19 17:02
   工作时长: 9小时4分
   ```

显示最近的打卡记录：

### ⚠️ 打卡注意事项

- **准时打卡**：
  - 不要忘记打卡
  - 忘记打卡可能被记为旷工

- **补卡申请**：
  - 如忘记打卡，可以申请补卡
  - 需要班组长审批
  - 提供合理的缺卡原因

- **考勤数据用途**：
  - 用于计算加班费
  - 用于评估全勤奖
  - 用于员工考核

## 常见问题

### Q1: 怎样快速报工？

**A**: 
推荐使用**扫码报工**，最快：
1. 打开手机浏览器，进入员工端应用
2. 点击首页「扫码报工」或底部「报工」
3. 对准任务标签上的二维码
4. 输入合格数和不良数（2秒钟）
5. 点击「提交」
6. 完成！（整个流程不超过30秒）

---

### Q2: 报工被驳回了，怎么办？

**A**: 
1. 收到驳回通知消息后，查看驳回原因
2. 根据班组长的反馈，修改报工
3. 重新进入「我的任务」
4. 找到被驳回的任务
5. 重新报工（可以修正数据和照片）
6. 提交新的报工

常见驳回原因：
- 📸 照片不清晰→重新拍照
- 📊 数量异常→确认合理数量
- 💬 没有备注说明→添加说明
- 🎥 没上传视频→补上视频证据

---

### Q3: 报工多久能到账工资？

**A**: 
工资到账流程：
```
1. 你报工提交
   ↓ （立即）
2. 显示预估工资
   ↓ （通常当天）
3. 班组长初审 ✓/✗
   ↓ 如通过
4. QC 终审 ✓/✗
   ↓ 如通过
5. 自动计入月工资
   ↓ （月底）
6. 财务确认工资
   ↓ （通常月底或次月初）
7. 查看工资条
```

**时间表**:
- 报工 → 预估工资: 立即（秒级）
- 预估 → 最终工资: 1-3天（等审核）
- 月工资最终确认: 通常月底或次月初

---

### Q4: 预估工资 vs 最终工资，为什么不一样？

**A**: 
原因包括：
1. **报工被驳回**：返工或废品不计工资
2. **还在审核中**：待初审或待终审的报工还没计入
3. **补贴调整**：最终工资可能加入补贴或扣款
4. **工价调整**：新月份的工价可能有变化

所以：**预估工资通常 > 最终工资**

---

### Q5: 如何避免报工被驳回？

**A**: 
几个关键点：
1. **数字准确**：合格数和不良数要真实，不要虚报
2. **拍好照片**：至少3张，清晰展示产品细节
3. **写好备注**：有问题一定说明（如材料不符、机器故障等）
4. **及时报工**：不要积压，当天完成当天报
5. **咨询班组长**：不确定时先问班组长，比返工更快

---

### Q6: 迟到会怎样？

**A**: 
迟到影响：
```
迟到15分钟内  → 通常不扣款（依工厂规定）
迟到15-30分钟 → 扣款¥5-10（通常）
迟到30分钟以上 → 扣款¥10-20或按小时扣
累计迟到3次    → 丧失全勤奖（¥50）
```

**建议**:
- 提前15分钟到达工位
- 不要踩点
- 偶尔迟到不要紧，避免频繁迟到

---

### Q7: 忘记打卡怎么办？

**A**: 
1. **及时补卡**：
   - 进入考勤打卡页面
   - 点击「补卡申请」
   - 选择补卡日期和时间
   - 填写补卡原因
   - 提交申请

2. **等待班组长审批**：
   - 班组长在 1-2 小时内审批
   - 审批通过后自动补上考勤记录
   - 审批不通过可能记为缺卡

3. **避免缺卡**：
   - 养成习惯，上班和下班都打卡
   - 可以设置手机闹钟提醒

---

### Q8: 工资条里的金额不对，怎么办？

**A**: 
1. **先自己核对**：
   - 看看是否有报工被驳回
   - 看看补贴或扣款是否正确
   - 对比之前月份的工资

2. **询问班组长**：
   - 最快的方式
   - 班组长了解你的工作情况
   - 能解释异常的原因

3. **联系财务**：
   - 如班组长也解释不了
   - 直接向财务提出异议
   - 附上错误的具体位置

4. **不要签名**：
   - 在异议解决前，不要签名确认
   - 签名后就无法修改了

---

### Q9: 报工上传的照片或视频丢失了，怎么办？

**A**: 
1. **若还未提交**：
   - 可以重新拍照和录视频
   - 删除旧的，选择新的文件

2. **若已提交**：
   - 照片/视频已保存在系统
   - 班组长可以查看
   - 如需补充证据，可以通过备注说明

3. **网络问题导致上传失败**：
   - 等网络恢复后重试
   - 或改用 WiFi 上传
   - 文件较大时可压缩后上传

### Q11: 应用崩溃或无法连接怎么办？

**A**: 
1. **刷新页面**：
   - Ctrl+R （Windows）或 Cmd+R （Mac）
   - 或点击浏览器刷新按钮

   - 重新打开应用并登录

4. **检查网络**：
   - 确保手机已连接 WiFi 或 4G
   - 尝试打开其他网页测试网络

5. **联系 IT 部门**：
   - 如问题仍未解决
   - 联系工厂 IT 或技术支持

## 🎯 员工使用建议

### ✅ DO（应该做）

- ✅ **每天登录检查**：查看是否有新任务派工
- ✅ **及时报工**：当天完成当天报工，不要积压
- ✅ **拍好照片**：清晰的质量证据，班组长审核更快
- ✅ **准时打卡**：签到和签退，避免迟到和缺卡
- ✅ **核实工资**：每月查看工资条，及时发现问题
- ✅ **定期修改密码**：每月至少一次，保护账户

### ❌ DON'T（不应该做）

- ❌ **不要虚报数量**：合格数/不良数要真实，虚报影响工资和信誉
- ❌ **不要忘记打卡**：可能被记为旷工，扣款
- ❌ **不要频繁刷新**：不会加快速度，只会浪费流量
- ❌ **不要在公共设备保持登录**：安全隐患，容易泄露个人信息
- ❌ **不要忽视驳回通知**：驳回意味着报工无效，需要重新报工
- ❌ **不要告诉他人密码**：账户安全很重要
