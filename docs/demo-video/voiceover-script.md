# LightMes Demo 视频配音录制脚本

> 给配音员/AI 配音使用
> 包含：每句台词、停顿位置、重音标记、情感提示、气息标记
>
> 录制格式建议：WAV 48kHz 24bit
> 总时长：约 140 秒（留 10 秒缓冲）

---

## 配音员要求

| 项目 | 要求 |
|------|------|
| 音色 | 男声,沉稳,有亲切感,避免过于播音腔 |
| 年龄感 | 30-45 岁,有「邻家老板」感 |
| 语速 | 280-300 字/分钟（正常偏慢） |
| 音量 | -3dB 到 -6dB 之间,留足动态空间 |
| 情感 | 段 1-2 略带疲惫和焦虑,段 3 转明朗,段 4 干练流畅,段 5 数字加重,段 6 温和邀请 |
| 收音 | 建议小房间/录音棚,无明显回音/底噪 |

---

## 标记说明

| 标记 | 含义 |
|------|------|
| `【停顿 X 秒】` | 严格停顿 X 秒 |
| `<强>` | 重读,音量 +3dB |
| `<弱>` | 轻读,音量 -3dB |
| `<慢>` | 语速放慢 30% |
| `<快>` | 语速加快 20% |
| `（吸气）` | 明显吸气声,用于换气/转情绪 |
| `（叹气）` | 自然叹气声,营造疲惫感 |
| `<上扬>` | 尾音上扬,疑问/邀请语气 |
| `<下沉>` | 尾音下沉,陈述/终结语气 |
| `→` | 数字连读（如「3 天→10 分钟」中间不停） |

## 段 1：钩子（0:00-0:15,15 秒）

### 第 1 句（0:00-0:08）

```
（叹气声,1 秒,自然叹气,带疲惫）
<慢>王<强>厂<强>长,
【停顿 0.5 秒】
<弱>又<慢>在<慢>加<慢>班<慢>啊?
<下沉>
```

**情感提示**：叹气声要自然,不要做作。「王厂长」要叫人感,「又在加班啊」带点调侃和心疼。

**技术提示**：叹气声单独录一条,后期混入。叹气声不要盖过台词。

## 录音技术规范

### 麦克风设置
- 类型：电容麦（如 Blue Yeti / Rode NT1 / AT2020）
- 距离：15-20cm
- 防喷罩：必备
- 监听耳机：必备

### 录音参数
- 采样率：48kHz
- 位深：24bit
- 格式：WAV（无损）
- 单声道

### 录音环境
- 房间：3-5 平方米小房间,避免空旷
- 墙体：建议铺吸音棉或挂毯
- 关闭：空调、风扇、冰箱（避免底噪）
- 桌面：避免敲击声,垫软布

### 后期处理流程
1. 降噪（iZotope RX / Adobe Audition）
2. 去除齿音（de-esser）
3. 音量归一化（-3dB 到 -6dB）
4. 加混响（轻微,模拟「录音棚感」）
5. 拼接 + 节奏对齐画面
6. 背景音乐混音（-20dB）
7. 总输出：WAV → MP3（128kbps 或 192kbps）
