## 剪辑技术规范

### 剪辑节奏
- 段 1-2（痛点）：硬切,每镜 3-5 秒,制造紧迫感
- 段 2-3：闪白 + Logo 缩放,情绪转折
- 段 3-4：推进过渡,自然流畅
- 段 4 内部：硬切,但单镜内动作连贯
- 段 4-5：闪黑 + 数字飞入
- 段 5-6：数字汇聚 + CTA 渐入

### 音频混音
| 层 | 音量 | 说明 |
|-----|------|------|
| 配音主轨 | -3dB | 主声轨,清晰可辨 |
| 背景音乐 | -20dB | 衬托,不抢戏 |
| 环境音/音效 | -15dB | 段 1-2 突出实拍环境感 |

### 字幕规范
- 字体：思源黑体 Bold
- 字号：48-56 像素（1080p 横屏）
- 描边：2 像素黑色
- 位置：画面底部 1/3 区域,左右居中
- 行数：最多 2 行
- 关键词高亮：「3 秒」「10 分钟」「200 件」用黄色或红色高亮

### 色彩分级
- 实拍段（段 1-2）：暖黄偏冷,带压抑感
- UI 录屏段（段 3-4）：清新蓝绿,对比度+10%
- 对比段（段 5）：高对比黑白红
- CTA 段（段 6）：暖色调,亲近感
