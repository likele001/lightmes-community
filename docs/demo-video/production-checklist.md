# LightMes Demo 视频拍摄制作执行手册

> 本文档是 Demo 视频的「施工图」
> 项目经理、摄影师、剪辑师、音效师都可参考
> 制作风格：B 实拍车间 + 录屏穿插
> 目标周期：7 天交付

---

## 一、项目总览

### 项目基本信息

| 项目 | 内容 |
|------|------|
| 项目名称 | LightMes 2 分 30 秒 Demo 视频 |
| 用途 | 官网首屏、B 站、视频号、销售提案 |
| 时长 | 2 分 30 秒（150 秒） |
| 分辨率 | 1920x1080（横屏主版） / 1080x1920（竖屏衍生版） |
| 帧率 | 30 fps |
| 风格 | 实拍 + 录屏穿插 + 后期动画 |
| 周期 | 7 个工作日 |
| 预算 | ¥0-3000（自做）/ ¥5000-15000（外包） |

### 项目里程碑

| 阶段 | 时间 | 交付物 |
|------|------|--------|
| D1 脚本定稿 | Day 1 | 完整逐字稿、分镜表、配音脚本、SRT |
| D2-D3 素材采集 | Day 2-3 | 实拍素材、录屏素材、配音文件 |
| D4 后期初剪 | Day 4 | 初剪版本（含音视频） |
| D5 终剪调色 | Day 5 | 终剪版本（含调色、音效、字幕） |
| D6 多版本输出 | Day 6 | 2 分 30 秒 + 1 分钟 + 30 秒 三个版本 |
| D7 审片定稿 | Day 7 | 最终定稿、发布素材包 |

- 标题：[LightMes 工厂管理系统] 月底算工资 3 天 → 10 分钟，老板必看！

- 弹幕引导：开 5 秒「老板视角」弹幕
