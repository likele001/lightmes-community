# LightMes Demo 视频完整逐字稿

> 视频类型：产品宣传 Demo
> 总时长：2 分 30 秒（150 秒）
> 拍摄风格：B 实拍车间 + 录屏穿插（推荐）
> 配音方式：AI 配音（剪映专业版「磁性男声-沉稳」/ 真人录制亦可）
> 适用渠道：官网首屏、B 站、视频号、销售提案
>
> 文案原则：少说名词多说场景,每个数字都可量化,老板视角切入
