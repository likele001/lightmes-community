# LightMes 进阶能力说明

> 版本：`advanced-features-20260529`  
> 涵盖：甘特图拖拽、WebSocket 大屏推送、国际化（i18n）、GPS 考勤围栏

---

## 一、功能列表

| 功能 | 说明 | 入口 / API |
|------|------|------------|
| **甘特图拖拽** | PC 生产计划甘特 Tab，拖拽计划条整体平移起止日期 | 计划 → 生产计划 → 甘特图；`PUT /admin/plans/{id}` |
| **WebSocket 推送** | 车间大屏优先 WebSocket 刷新，失败降级 15s 轮询 | `WS /api/ws/dashboard?token=` |
| **国际化 i18n** | 管理端中英文切换（菜单/布局/甘特提示） | 顶栏 **中 / EN** |
| **GPS 考勤** | H5 打卡自动采集经纬度；可选租户地理围栏 | H5 考勤；`POST /h5/attendance/check-in` |
| **地理围栏配置** | 管理员设置打卡中心点与半径 | `GET/PUT /admin/attendance-records/geofence` |

---

## 二、甘特图拖拽

1. 进入 **生产计划 → 甘特图**（PC 宽屏）
2. 鼠标按住蓝色计划条左右拖动
3. 松开即保存新的 `start_date` / `end_date`（按天对齐）

注意：仅平移整段计划，暂不支持拖拽改长度；移动端仍为折叠卡片视图。

---

## 三、WebSocket 大屏

- 连接：`wss://域名/api/ws/dashboard?token=<JWT>`
- 服务端每 **15 秒** 推送 `{"type":"refresh","channel":"dashboard"}`
- 客户端收到后重新拉取 summary/charts/kanban

**Nginx 需开启 WebSocket 反代**（宝塔站点配置 → 反向代理 → 开启 WS）：

```nginx
proxy_http_version 1.1;
proxy_set_header Upgrade $http_upgrade;
proxy_set_header Connection "upgrade";
```

## 四、国际化（i18n）

- 框架：`vue-i18n`（`frontend-admin-pro`）
- 语言包：`src/locales/zh-CN.ts`、`en-US.ts`
- 切换后页面刷新以同步 Element Plus 组件语言
- 当前已翻译：**全部侧栏子菜单**、顶栏、甘特提示、考勤记录页（含围栏配置）
- 切换语言：顶栏 **中 / EN**（刷新后 Element Plus 组件语言同步）

## 五、GPS 考勤与地理围栏

### 5.1 员工 H5 打卡

- 打卡时自动调用浏览器定位 API，提交 `latitude` / `longitude`
- 若租户未启用围栏，有定位则记录，无定位也可打卡（与旧行为兼容）

### 5.2 管理员配置围栏

**管理端 UI**：系统管理 → **考勤记录** 页顶「GPS 打卡围栏」卡片，可开关、填中心经纬度与半径并保存。

或使用 API（需 `attendance.manage`）：

```http
PUT /api/admin/attendance-records/geofence
{
  "enabled": true,
  "lat": 31.2304,
  "lng": 121.4737,
  "radius_m": 200
}
```

启用后：员工必须在半径内打卡，否则返回 400 及距离提示。

### 5.3 数据库

迁移 `0044_attendance_gps` 为 `attendance_records` 增加：

- `check_in_lat` / `check_in_lng`
- `check_out_lat` / `check_out_lng`
