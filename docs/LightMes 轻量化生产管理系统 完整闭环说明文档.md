# LightMes 轻量化生产管理系统 完整闭环说明文档

**文档版本**：2026\-05\-25 自动化重构版

**系统定位**：面向中小型加工厂的轻量化、可快速交付的生产管理系统

**系统架构**：多租户SaaS架构 \| 后端FastAPI \+ 前端Vue3三端 \+ 微信小程序

# 一、系统全景架构

## 1\.1 访问层级架构

### 访客/公开访问层

前端门户（frontend\-portal）：支持企业注册、套餐定价、在线支付功能；H5溯源页 /trace 支持成品追溯公开查询，无权限门槛，对外透明展示产品生产信息。

### 终端用户访问层

面向企业内部员工、管理员及客户，提供三端一体化访问能力，覆盖全场景操作需求：

- **前端管理后台（admin\-pro）**：基于Vue3 \+ Element搭建，适配电脑端全功能运维、业务管控、数据查看

- **H5移动端（frontend\-h5）**：基于Vue3 \+ Vant 4搭建，适配手机浏览器，满足移动办公、报工、查薪、客户自助操作

- **微信小程序（miniapp）**：基于uni\-app开发，轻量化便携，聚焦员工报工、考勤、管理员移动端审核看盘核心场景

### 后端服务层

基于FastAPI构建分层架构，包含API路由、业务逻辑、CRUD数据操作、数据模型四层核心模块，实现所有业务接口的统一调度与处理。

### 数据与存储底层

支撑系统所有业务数据存储、缓存与文件管理，核心组件如下：

- MySQL 5\.7\+ 数据库：核心业务数据存储，通过Alembic完成43个版本迭代迁移，数据结构稳定可迭代

- Redis：承担系统缓存、Celery任务代理与任务结果存储工作

- 文件存储：支持本地存储、阿里云OSS、腾讯COS、七牛Kodo多驱动适配，灵活适配不同部署场景

# 三、业务核心闭环流程

## 3\.1 主生产全流程闭环

产品基础定义 → 型号（SKU）配置 → 工序定义 → 工艺路线编排 → 工序工价配置 → 客户自助/后台手工录入订单 → 订单确认 → 系统自动拆解SKU级工单 → 生产计划排产（甘特图拖拽/OR\-Tools自动排程）→ 物料齐套检查（BOM缺料预警）→ 按员工技能\+负荷自动派工 → 释放生产计划、生成工序任务 → 员工扫码/手工报工（逐件报工为主）→ AI预审（免审/抽检/全检判定）→ 班组长初审 → QC终审（驳回则记录原因、重新报工）→ 审核通过自动生成工资明细 → 成品赋唯一追溯码 → 发货出库 → 客户公开溯源查询，形成生产全链路闭环。

## 3\.2 工资计算闭环

报工完成且QC终审通过 → 系统自动生成SalaryItem工资明细（计算公式：员工ID×SKU ID×工序ID×单件工价×合格数量=单品薪资）→ 月度自动汇总所有薪资明细 → 生成SalarySlip月度工资条（明细总薪资\+补贴金额\-扣款金额=实发工资）→ 员工通过H5/小程序查看工资条并电子签名确认 → 支持Celery异步导出Excel工资表，全程系统自动核算，杜绝人工篡改，保障薪资数据严谨性。

## 3\.3 全流程溯源闭环

产品首工序报工审核通过 → 系统自动生成成品追溯码（FP\+租户ID\+递增序号），初始化TraceCode溯源记录 → 后续每道工序报工完成后，自动关联父追溯码，生成全工序流转溯源记录 → 产品成品发货，绑定发货单据信息 → 客户/终端用户通过H5公开溯源页扫码查询 → 可完整查看产品基础信息、全工序流转轨迹、操作人员、生产时间、各工序多媒体佐证、订单信息、上游物料批次，实现从物料到成品、从生产到售后的全维度溯源闭环。

## 3\.4 CRM销售闭环

客户档案录入、标签分类管理 → 生成销售机会（状态：潜在→资质审核→方案报价→洽谈中→成交/关闭）→ 机会赢单后自动生成销售订单 → 未及时跟进的销售机会由Celery定时任务自动回收至客户公海池 → 公海资源可重新分配跟进，形成客户资源、销售机会、订单转化的完整销售闭环，盘活闲置客户资源。

# 四、多租户SaaS架构设计

## 4\.1 平台运营侧能力

由平台管理员（PlatformUser）统一管控，支撑系统商业化运营与全局配置：

- 套餐管理：自定义SaaS套餐价格、时长、功能权限、用户配额，实现按需定价

- 租户管理：支持企业租户创建、启用、停用、自定义域名配置

- AI配置：平台级统一LLM模型接入与参数配置，全局赋能各租户

- 支付对接：打通在线支付通道，统一管理企业订阅订单

## 4\.2 租户隔离机制

所有业务数据表均配置**tenant\_id**租户外键，搭配（租户ID\+业务唯一键）联合唯一约束，实现租户数据物理隔离、权限完全独立。单租户内部可自主完成员工、角色、权限、部门管理，自定义系统字典、打印模板、自动化规则，所有业务模块独立运行，互不干扰，保障中小型企业数据安全与系统独立性。

# 五、AI智能中心能力

系统内置全方位AI智能模块，所有能力统一集成于backend/app/services/ai/目录，通过Celery Beat定时触发或主动调用，赋能生产全流程智能化升级。

|AI模块|核心功能说明|
|---|---|
|预警扫描（alerts\.py）|定时扫描逾期订单、生产低良率、设备运行异常等问题，自动推送系统通知，提前规避生产风险|
|每日简报（daily\_brief\.py）|自动生成厂长日报，汇总当日生产产值、订单达成率、逾期订单、生产异常等核心经营数据|
|排产优化（planning\_optimizer\.py）|基于OR\-Tools CP\-SAT约束求解算法智能排产，结合LLM给出优化建议，提升排产合理性与产能利用率|
|AI预审（audit\_prescreen\.py）|基于历史生产良率趋势、员工报工数据，智能判定报工单据免审、抽检、全检，大幅降低审核人力成本|
|定价AI（pricing\_ai\.py）|分析历史工序工价、生产成本数据，智能推荐合理工序单价，辅助企业定价决策|
|数字孪生（digital\_twin\.py）|实时映射车间生产状态、设备运行、订单进度，实现生产场景数字化可视化管控|
|设备健康（equipment\_health\.py）|基于设备运行数据、点检记录，预测设备维护周期，智能识别运行异常，降低设备故障停机风险|
|因果推断（causal\_inference\.py）|对产品不良品数据进行推演分析，定位生产缺陷核心原因，辅助工艺优化|
|视觉质检（visual\_quality\.py）|通过图片视觉识别技术，智能检测产品外观质量问题，辅助人工质检|
|RAG帮助（rag\_help\.py）|搭建系统知识库，支持用户智能问答，快速解答系统操作、业务流程相关问题|
|厂场景（scenes\.py）|基于LLM实现生产场景智能编排与数据分析，适配不同工厂个性化生产场景需求|
