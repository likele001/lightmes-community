# LightMes 小程序使用指南

> 工程：`lightmes-miniapp/`（**uni-app**，非原生 miniprogram）  
> 产品方案：[`小程序方案.md`](./小程序方案.md)

## 前提

- Node.js >= 18
- 微信开发者工具
- 运行中的 LightMes FastAPI 后端
- 微信小程序 AppID 与 `WX_MINIAPP_APPID` / `WX_MINIAPP_SECRET`

## 开发

```bash
cd lightmes-miniapp
npm install
cp .env.development .env.local   # 可选，修改 VITE_API_BASE_URL
npm run dev:mp-weixin
```

微信开发者工具 → 导入项目 → 目录选择：

`lightmes-miniapp/dist/dev/mp-weixin`

## 生产构建

```bash
npm run build:mp-weixin
```

发布目录：`dist/build/mp-weixin`

## 登录与角色

1. **须先填租户编码**（如 `DEMO`），与 PC/H5 登录一致。
2. **微信一键登录**：用微信 openid 绑定到 **本租户下某个系统用户**（`users` 表的一条记录）。
   - 绑的是 **用户名账号**，不是「H5 专用」或「管理端专用」。
   - 用 **员工账号** 绑定 → 小程序走员工端（报工、任务等）。
   - 用 **厂长/管理员账号** 绑定 → 可按角色进管理端；若兼有 `leader` 且带员工能力，可切换员工端。
3. **账号密码登录**：与 H5 相同接口，不经过微信绑定。
4. 同时具备员工与管理权限时，进入 **模式选择页**。

### 和 H5、管理端的关系

| 端 | 登录方式 | 账号来源 |
|----|----------|----------|
| PC 管理端 | 租户 + 用户名 + 密码 | 同一 `users` |
| 手机 H5 | 租户 + 用户名 + 密码 | 同一 `users` |
| 微信小程序 | 微信绑定 或 租户+用户名+密码 | 同一 `users` |

一个微信只能绑定 **一个** 系统用户；解绑需管理员在数据库清空该用户的 `wx_miniapp_openid`（后续可做后台解绑功能）。

## AI 功能（Phase E，需平台开启 AI 且后端已部署）

| 端 | 入口 | 权限 | 说明 |
|----|------|------|------|
| **员工端** | 逐件报工页 → **AI 检查一下** | `ai.report_assist` | 提交前检查数量/备注，与 H5 一致 |
| **管理端** | 生产计划编辑 → **AI 排产建议** | `ai.use` + `plan.manage` | **只读**：LLM / OR-Tools 双 Tab，不自动改计划 |
| **管理端** | 件次审核列表 → **AI 批量摘要** | `ai.use` + `report.audit` | 审核 Tab 与件次列表页均可入口 |
| **管理端** | 件次详情 → **Vision 识图辅助** | 同上 | 需平台配置 Vision 模型 + 员工已上传照片 |
| **管理端** | 首页 → **AI 数据预警** | `ai.alert.view` | 最近 5 条；有 `setting.manage` 可改阈值 |
| **管理端** | 首页快捷入口 | 见权限 | **工厂助手**、**智能帮助**、**产能设置** |
| **管理端** | 系统设置 → **AI 网关** | `ai.use` + `setting.manage` | 租户独立 Base URL / API Key |
| **管理端** | 生产管理 → **产能设置** | `plan.manage` | 计件/工时、默认/人员/车间产能（件/天） |
| **管理端** | 我的 → **工厂助手** | `ai.use` | 老板数据问答（非流式） |
| **员工/管理** | 我的 → **智能帮助** | 登录即可 | RAG 操作帮助 |

| 端 | 入口 | 说明 |
|----|------|------|
| **管理端** | 功能 Tab → **智能中心** | 工厂助手、帮助、深度分析、统计、**生产自动化** |

AI 接口超时 **120 秒**，请确保微信公众平台 request 合法域名指向的后端 Nginx `proxy_read_timeout ≥ 120s`。

更新 AI 相关页面后需重新构建并上传：

```bash
cd /www/wwwroot/lightmes/lightmes-miniapp
npm install
npm run build:mp-weixin
```

## 常见问题

**Q: 编译报 pages.json duplication？**  
A: `pages` 必须使用对象格式 `{ path, style }`，不能用纯字符串。

**Q: 管理接口 403？**  
A: 确认账号非纯 `employee` 角色，且拥有对应 `permission`；请求会自动带 Admin Portal 头。

**Q: 报工提交失败？**  
A: 逐件任务须走逐件报工页；检查是否已派工、数量是否超限。

**Q: AI 检查一下报超时或 503？**  
A: 确认平台 AI 总开关与默认模型已配置；后端已重启；合法域名与 Nginx 超时 ≥ 120s。
