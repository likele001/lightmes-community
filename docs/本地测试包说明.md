# LightMes 本地测试包说明

> 完整包含：**后端源码 + 前端源码（含 package.json）+ 已构建 dist**，适合本机无域名快速验证。

## 一、包内结构

```text
lightmes/
├── backend/                 # FastAPI 后端
│   ├── env.example          # 环境变量模板（宝塔可见，复制为 .env）
│   └── .env.example         # 同上（隐藏文件）
├── frontend-admin-pro/      # 管理端源码 + dist/
├── frontend-h5/             # H5 源码 + dist/
├── docs/
└── start.sh
```

## 二、环境要求

| 组件 | 版本 |
|------|------|
| Python | 3.10+ |
| MySQL | 5.7+ |
| Node.js | 18+（跑 `npm run dev` 需要） |
| Redis | 可选（Celery） |

## 三、本地无域名访问（推荐）

**不用配域名**，全部用 `127.0.0.1` + 端口。

### 1. 后端

```bash
cd backend
cp env.example .env
# 编辑 .env：DB_URL、JWT_SECRET 等

pip install -r requirements.txt
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

### 2. 管理端（开发模式，自动代理 /api）

```bash
cd frontend-admin-pro
npm install
npm run dev
```

浏览器：**http://127.0.0.1:5173/t/DEMO/login**（admin / admin123）

平台总控：**http://127.0.0.1:5173/platform/login**（platform / platform123）

### 3. H5（可选）

```bash
cd frontend-h5
npm install
npm run dev -- --port 5174
```

浏览器：**http://127.0.0.1:5174**

### 4. `.env` 本地示例（无域名）

```env
DB_URL=mysql+pymysql://root:密码@127.0.0.1:3306/lightmes?charset=utf8mb4
JWT_SECRET=local_test_secret
STORAGE_DRIVER=local
STORAGE_LOCAL_ROOT=./data/storage
PUBLIC_BASE_URL=http://127.0.0.1:5173
H5_PUBLIC_BASE_URL=http://127.0.0.1:5174
```

## 四、只用 dist、不跑 npm dev

包内已有 `frontend-admin-pro/dist`，但静态站无法自己转发 API，需本机 Nginx 把 `/api` 反代到 `8000`，或继续用上面的 **npm run dev**（最简单）。

重新构建：

```bash
cd frontend-admin-pro && npm install && npm run build
cd frontend-h5 && npm install && npm run build
```

## 六、常见问题

- **找不到 package.json**：请用完整包（含 frontend-admin-pro 整个目录），不要只有 dist 的精简包
- **找不到 .env.example**：用 `backend/env.example` 复制为 `.env`（宝塔隐藏点开头的文件）
- **npm install 慢**：可换国内镜像 `npm config set registry https://registry.npmmirror.com`
