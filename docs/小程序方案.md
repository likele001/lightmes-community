# LightMes 微信小程序方案（uni-app v2）

> 工程：`lightmes-miniapp/` · 技术栈：uni-app + Vue3 + TS + Pinia + vue-i18n（客户端）  
> 范围：员工端 + PC 租户管理端 + **客户自助端**（三端模式）；不含 SaaS 平台运营端

## 1. 架构

```
主包
├── shared/login、bind-account、role-select、trace（公开溯源）
├── tabs/tab-0..4（自定义 TabBar，员工/管理双模式切换）
└── customer/tab-0..2（客户端独立 Tab + CustTabBar 组件导航）

分包 pages-customer（5 页）
└── 订单详情/进度/发货/售后、对账详情（对齐 frontend-h5 客户门户）

分包 pages-employee（8 页）
└── 报工/任务/工资/消息等（对齐 frontend-h5）

分包 pages-admin（56 页）
└── 对齐 frontend-admin-pro 全部租户模块
```

### 三端模式对照

| 端 | 登录 Portal | 主界面 | 底部导航 |
|----|-------------|--------|----------|
| 员工端 | 员工端 | `pages/tabs/tab-*` | 微信 custom-tab-bar |
| 管理端 | 管理端 | `pages/tabs/tab-*` | 微信 custom-tab-bar |
| **客户端** | **客户端** | **`pages/customer/tab-*`** | **CustTabBar（reLaunch 切换）** |

客户端与员工/管理 **完全隔离**：不复用 `pages/tabs`，不修改微信原生 `tabBar.list`。

## 2. 权限与安全

| 层级 | 实现 |
|------|------|
| 登录 | `POST /miniapp/auth/login` + `GET /auth/me` |
| 员工 API | `/h5/*`，角色 `employee` / `leader` |
| 客户 API | `/h5/customer/*`，角色 `customer` |
| 公开溯源 | `GET /h5/public/trace/{code}`，无需登录 |
| 管理 API | `/admin/*`、`/dashboard/*`，请求头 `X-LightMes-Portal: admin` |
| 前端守卫 | `usePermission()` + `permissions[]` 控制菜单与页面 |
| 多角色 | 可在「员工 / 管理 / 客户端」间切换（`role-select`） |

权限码定义：`src/constants/permissions.ts`（与 `backend/app/core/seed.py` 一致）。

## 3. 功能对照

### 员工端（100% 对齐 H5）

| 页面 | 路径 |
|------|------|
| 首页/任务/报工/打卡/我的 | Tab `tab-0`..`tab-4` |
| 扫码报工 | `pages-employee/report/scan` |
| 逐件报工 | `pages-employee/report/unit` |
| 工资/工资条签名 | `pages-employee/salary/*` |
| 飞书/企微通知绑定 | 员工「我的」`ChannelBindCard` |

### 客户端（对齐 H5 客户门户）

| 页面 | 路径 |
|------|------|
| 下单（产品浏览 + 我的订单） | `pages/customer/tab-0` |
| 对账单列表 | `pages/customer/tab-1` |
| 我的（语言切换、溯源入口） | `pages/customer/tab-2` |
| 订单详情 | `pages-customer/order/detail` |
| 生产进度 | `pages-customer/order/progress` |
| 发货物流 | `pages-customer/order/shipments` |
| 售后申请 | `pages-customer/order/after-sales` |
| 对账详情（确认/已付/CSV） | `pages-customer/statements/detail` |

**多语言**：客户端支持 zh-CN / en-US / ko-KR（`src/locales/`），员工/管理端保持中文。

**通知绑定**：客户端「我的」预留 `CUSTOMER_CHANNEL_BIND_SLOT`，本期不实现飞书/企微绑定。

### 公开溯源

| 页面 | 路径 | 说明 |
|------|------|------|
| 产品追溯 | `pages/shared/trace/index` | 无需登录；支持 `?code=` 及小程序码 `scene` 参数 |

入口：登录页底部、员工「我的」、客户端「我的」。

### 管理端（对齐 PC）

系统、主数据、生产、CRM、计划、仓储、设备、采购、财务、报表、看板等 56 个列表/详情页，采用 **MListLayout** 移动列表范式。

## 4. 部署

1. `npm install && npm run build:mp-weixin`
2. 微信开发者工具打开 `dist/build/mp-weixin`
3. 配置合法域名、`VITE_API_BASE_URL`、小程序 AppID
4. 后端 `WX_MINIAPP_*` 环境变量
5. 溯源媒体域名：`/h5/public/trace/media/{id}` 需加入 download/request 白名单
